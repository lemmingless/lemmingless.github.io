1.1 关于梯度

pytorch\chapter_preliminaries\autograd.ipynb中的问题

tensor([ 0., 4., 8., 12.])

梯度的定义如下公式$$grandf(x_{ 0 },x_{ 1 },…,x_{ n } )=(\frac{ \partial f }{ \partial x_{ 0 } } ,…,\frac{ \partial f }{ \partial x_{ j } },…,\frac{ \partial f }{ \partial x_{ n } })$$

得到输出结果$tensor([0.,4.,8.,12.])$的计算过程为$y=2XX^{T}$，$X=(x_{1},x_{2},x_{3},x_{4})$，那么$y$可以写成$y=2(x_{1}^{2}, x_{2}^{2},x_{3}^{2},x_{4}^{2})$，按上面的梯度公式，$$\frac{\partial y}{x_{1}} =4x_{1},\frac{\partial y}{x_{2}} =4x_{2},\frac{\partial y}{x_{3}} =4x_{3},\frac{\partial y}{x_{4}} =4x_{4}$$代入$X=(0,1,2,3)$，得到结果$x.grad=tensor([0.,4.,8.,12.])$

同理，下面代码中，$y=x.sum()$表示着$y=x_{1}+x_{2}+x_{3}+x_{4}$，再按梯度公式，得到结果$x.grad()=tensor([1.,1.,1.,1.])$
